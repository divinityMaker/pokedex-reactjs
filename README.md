### Pre requirements
- [Node.js](https://nodejs.org/en/). 


### Run the project

```bash
# Clone this repository
$ git clone https://github.com/divinityMaker/pokedex-reactjs

# Get into the project folder
$ cd pokedex-reactjs

# Install the dependencies
$ npm install

# Execute the app in dev mode
$ npm run dev

```
